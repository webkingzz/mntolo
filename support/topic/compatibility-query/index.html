    Hi,
    For the dropdown filter, I added this css:

    .select2-results__option {color:#000 !important; }

    For the related products text, I added this css:

    .related.products ul.products h2.woocommerce-loop-product__title {
font-size: 14px !important; 
}

    Please clear your browser cache and check.
    For the shop page error, I believe that an option needs to be selected at Woocommerce > Woocommerce Toggle Shop Products

    Hi,
    I found that changing the “Query type” in the widget changes how it looks, The one you like is the Query type: AND
    while the other is: OR

    You can read about this here: Filter Products by Attribute
    You said they were the same before the update, so perhaps this is what changed accidentally.
    Please try changing this option.

    Mike
