    Hi!

    1. Try with this:

    #contentwrap *{
    color: red;
}

    2. You can use pseudo-classes for that:

    a{
color: red;
}
a:active{
color: blue;
}
a:hover{
color: green;
}

    3. The following code will do it but you’d need to edit the background image in order to match the new height otherwise it will look like this.

    #top .logo a img {
    width: 1000px;
}

.catnav_wrapper {
    margin-top: 125px;
}

    4. This will do it:

    #contentwrap {
    background: #010101;
    overflow: hidden;
}

    Best regards,
    Josue
