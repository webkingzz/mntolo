    Hi there, I have a big issue with the latest WP update in combination enfold. I can’t find my theme options anymore!
    Pleaase can you help me
    Regards Nieck Jan Boschman

    September 30, 2020 at 4:37 am #1249553

    Nikko
    Moderator

    Hi Nieck Jan Boschman,

    WordPress 5.5 have removed jQuery Migrate file from its core which is used by Enfold Theme before 4.7.6.2 which causes the issue, to temporarily fix this, please install Enable jQuery Migrate Helper then afterwards backup your site and upgrade Enfold to the latest version (4.7.6.3), then you should be able to safely remove the plugin afterwards unless there are plugins that requires it as well.

    Best regards,
    Nikko
