    Hi,

    Please try this in Quick CSS:

    .main_menu a:hover, .main_menu .current-menu-item>a, .main_menu .current_page_item>a {
	background-color: #83ad1a!important;
}

    And for removing our copyright in the footer, please add this in Quick CSS as well:

    .copyright a:nth-child(2) {display:none;}
