    Fixed blurry thumbnails by adding this to the functions.php file:

    add_filter( ‘woocommerce_get_image_size_gallery_thumbnail’, function( $size ) {
    $size[‘width’] = 300;
    $size[‘height’] = 300;
    $size[‘crop’] = 1;
    return $size;
    }, 10, 1);
