    Hi,

    Enfold adds the default widgets if no widgets are added to the widget area, just put some widgets in there and it should be replaced. If you want to remove it, you can go to Enfold > Footer > Default Footer Widgets & Socket Settings and set this to Display only the socket (no footer widgets). Hope this helps :)

    Best regards,
    Nikko
