    Dude
    Moderator

    Hi,

    I looked into it and the reason is all (or most) of you pages are set to “Privately Published”. WordPress will not show these pages as parent pages or menu items. Please change the “Visibility” of these pages to “Public” and they’ll pop up in the menu editor too (I tested it with the entry “Acoustic Guitars” – http://sing-music.com/wp-admin/post.php?post=1749&action=edit ). If you don’t want that other website visitors can read these pages I’d recommend to activate a maintenance plugin like: https://de.wordpress.org/plugins/coming-soon/

    But, I’m trying to figure how to LINK the main category to the sub category. Basically, the main category to point to the sub category. I want when you hover over ‘Guitars’… Electric, Acoustic, etc. shows so that the customer sees the options for
    different guitars.

    Personally I’d add the main categories to the menu bar (as first level menu items). Then you could add the sub categories as second level menu items to the menu and Abundance will show them as simple dropdowns or mega menu (if you check the “Use as mega menu” box of the first level menu item) which fades in when the user hovers over the main category links.

    Best regards,
    Dude
