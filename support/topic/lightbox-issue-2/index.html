    Hi tattoolike,

    You’ll need to change the size of the image that gets generated by the functions.php file. Open up your functions.php file thats in the main folder of your theme files and look for this line:

    $k_option['custom']['imgSize']['L'] = array('width'=>610, 'height'=>260); // big preview pic for posts

    Change the dimensions for width and height to the max size you want to appear on hover. Keep in mind this will slow down your page loading if the images are too big.

    Regards,

    Devin

    Hi,

    Please try this type of code

    <img style="max-width: 400px; height: auto; " src="image.jpg" />

    Please install this plugin – Simple Image Sizes plugin ( http://wordpress.org/extend/plugins/simple-image-sizes/ ). The plugin will give you complete control over all image sizes in the theme – from aspect ratio, to width, whether to crop the image, and even has a built in thumbnail regenerator.

    After installing the plugin, you will find your Settings > Media page in the backend to contain the various sizes. Then simply find the dimensions you want to change. Save. Regenerate the images and see if you like what you see or want to tweak the image sizes further.

    Hi,

    Please go to Settings > Media in the Backend.

    Make the width 400. Make the height 9999 and make sure the checkbox is OFF that says ‘Crop thumbnail to exact dimensions (normally thumbnails are proportional)’

    Be careful allowing users to upload images. That’s the easiest way to get your website hacked when using incorrect code.

    Thanks,

    hi Nick,

    finally i resolve the problem changing the numbers of width and height from “pixels” in “auto” in function.php .

    #####################################################################

    # Define Thumbnail sizes

    #####################################################################

    $k_option = array(‘width’=>267, ‘height’=>180); // backend preview size, if changed does not affect the frontend

    $k_option = array(‘width’=>60, ‘height’=>60); // small preview pics, eg for sidebar news

    $k_option = array(‘width’=>180, ‘height’=>auto); // small preview pic for posts

    $k_option = array(‘width’=>auto, ‘height’=>auto); // big preview pic for posts

    $k_option = array(‘width’=>884, ‘height’=>390); // big images for fullsize pages

    Thank you for the advises !!!
